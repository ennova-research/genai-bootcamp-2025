# Generative AI Bootcamp

**Duration:** **6 Weeks** (8 hours per day → 4 h theory \+ 4 h labs)  
**Organizer:** Umana and Ennova Research – Data & AI XTDF Division  
**Track Focus:** Software Engineering for AI • Databases • LLM Agents  
**Tool Ecosystem:** Python • LangChain • FAISS • BigQuery • Vertex AI • Watson X

By the end, participants will:  

- Build production-ready AI applications in Python  
- Query databases and LLM pipelines seamlessly  
- Design and evaluate autonomous LLM agents  
- Deploy on Google Vertex AI and IBM Watson X  
- Present 2 end-to-end capstones as portfolio projects

## List of Topics

### Preliminaries (November 13-14)

**Theme:** Introduction to the Bootcamp.

* The AI Revolution
* Architecture of Enterprise AI apps
* Foundational model APIs

### Week 1 – Software Engineering for AI Applications (November 17-21)

**Theme:** Building robust Python backbones for AI systems.

* Python Refresher & Design Patterns
* OOP, Modules, Typing, Async Programming, API Calls, Services and Routing
* Design patterns for AI: Factory, Strategy, Adapter, Observer
* Logging, config management, error handling, testing (Mocking/PyTest)
* Packaging & virtual environments for MLOps
* Dockerization and Containers

### Week 2 – Databases & Data Pipelines for AI (November 24-28)

**Theme:** Efficient data handling for LLMs and retrieval systems.

* SQL fundamentals (SELECT, JOIN, AGGREGATE, VIEW)
* BigQuery basics and Python SDK integration
* Text-to-SQL and semantic querying with LLMs
* ETL pipelines and data cleaning for prompt engineering
* NoSQL and Vector Stores (FAISS, Chroma, Weaviate, Milvus, Pinecone, Redis)
* Graph databases  (Concepts with Neo4j and Cypher)

### Week 3 – Generative AI Paradigms (I): LLMs & Prompt Design (December 1-4)

**Theme:** Mastering core generative intelligence concepts.

* Transformers, attention, and context windows
* LLMs and SLMs 
* Prompt engineering (From zero-shot to chain-of-thought methods)
* Fine-tuning processes and methodologies (LoRA, PEFT)
* Prompt Engineering Guardrails AI framework
* Evaluation methods and metrics (Perplexity, Toxicity, Relevance, Precision, Drifting)

### Week 4 – Generative AI Paradigms (II): LLM Agents & Tool Use (December 9-12)

**Theme:** From LLMs to autonomous agents.

* LLMs and Agents, Routing and Query Analyzing
* LangChain and LangGraph agents, memory, tool invocation 
* Model Context Protocol (MCP) and A2A communication
* Retriever–Writer–Reviewer pattern
* Evaluation of agent reasoning and traceability

### Week 5 – Generative AI on the Cloud (December 15-19)

**Theme:** Deploying and scaling AI solutions with Vertex AI / Watson X

* Vertex AI Model Garden & Gemini APIs
* RAG chatbot using BigQuery \+ FAISS \+ Gemini
* Containerization (Docker) and serving via Cloud Run
* Monitoring and cost optimization
* Watson X AI Workbench overview \+ Prompt Tuning
* Integrating Watson X Foundation Models into Python apps
* Agent integration via LangChain connectors
* Performance and ethics evaluation
